WorkoutLogger is a full-stack MERN application that allows users to securely register, log in, and manage their workout routines. The platform enables authenticated users to create, view, update, and delete workout records, making it easy to track fitness progress over time.

Frontend: React.js, JavaScript, HTML5, CSS3
Backend: Node.js, Express.js
Database: MongoDB, Mongoose
Authentication: JWT(JSON Web Token)
Tools & Deployment: Git, GitHub, Render

Deployed_link : https://workoutlogger-frontend.onrender.com
